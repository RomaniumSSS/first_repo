# Meme Generator

A modern meme generator application with a redesigned UI featuring:
- **Left Sidebar**: Vertical meme template selection (Canva-style utility bar)
- **Main Canvas**: Large area displaying the selected meme
- **Bottom Utility Bar**: Rounded rectangle bar with editing tools

## How to Use

1. Open `index.html` in your web browser
2. Select a meme template from the left sidebar
3. Click "Text" in the bottom utility bar to add text
4. Click on the canvas to add text at that position
5. Click on existing text to select it
6. Drag selected text to move it
7. Use "Size" and "Color" buttons to customize text
8. Click "Download" to save your meme
